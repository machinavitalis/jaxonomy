# Tutorials

These notebooks introduce **Jaxonomy** step by step: block diagrams, simulation, customization, and optimization.

| Notebook | What you will learn |
|----------|---------------------|
| [Getting started](./01-getting-started.ipynb) | Build a simple diagram with `DiagramBuilder`, run `simulate`, inspect results. |
| [Creating custom blocks](./02-creating-custom-blocks.ipynb) | Implement your own `LeafSystem` blocks and wire them into diagrams. |
| [Creating custom acausal components](./03-creating-custom-acausal-components.ipynb) | Extend the acausal modeling layer with new components. |
| [Wrappers and decorators](./04-wrappers.ipynb) | Patterns for wrapping and decorating systems. |
| [Automatic differentiation and optimization](./05-automatic-differentiation-optimization.ipynb) | Use AD with models for optimization workflows. |

For more topic-specific notebooks (MPC, estimation, hardware-inspired demos, etc.), see **[Examples](../examples/index.md)**.

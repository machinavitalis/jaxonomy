# About

**Jaxonomy** is a Python simulation library focused on **composable block-diagram models**, **hybrid dynamics** (continuous + discrete + events), and a **JAX-first** execution path for performance and differentiation.

This documentation site is generated with **[MkDocs](https://www.mkdocs.org/)**, **[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)**, **[mkdocstrings](https://mkdocstrings.github.io/)** (API reference), and **[mkdocs-jupyter](https://github.com/danielfrg/mkdocs-jupyter)** (notebook pages). To build it locally from the repository:

```bash
pip install -r requirements.docs.txt
mkdocs build --clean
# or: mkdocs serve
```

General project information may appear on [www.jaxonomy.com](https://www.jaxonomy.com).
